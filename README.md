# Truss Master 🛠️

Un juego de puzzles de ingeniería estructural inspirado en "Truss Me!". Construye estructuras de madera estratégicas para soportar pesos masivos y evitar que toquen el suelo.

## 📱 Cómo Instalar en tu Móvil

Este juego es una **PWA (Progressive Web App)**, lo que significa que puedes instalarlo como una aplicación nativa siguiendo estos pasos:

### 1. Subir a GitHub (Usando GitHub Desktop)
1. Abre **GitHub Desktop**.
2. Selecciona `File` > `Add Local Repository` y elige esta carpeta.
3. Escribe un resumen (ej: "Primer commit") y dale a **Commit to main**.
4. Dale a **Publish repository** para subirlo a tu cuenta de GitHub.

### 2. Activar GitHub Pages
1. Ve a tu repositorio en la web de GitHub.
2. Ve a **Settings** (Ajustes) > **Pages**.
3. En **Build and deployment**, selecciona la rama `main` y la carpeta `/(root)`. Dale a **Save**.
4. Espera un minuto a que GitHub te dé una URL (ej: `https://tuusuario.github.io/tu-repo/`).

### 3. Instalar en el móvil
1. Abre esa URL en tu móvil (Safari en iOS, Chrome en Android).
2. **iOS**: Dale al botón de **Compartir** y selecciona **"Añadir a la pantalla de inicio"**.
3. **Android**: Dale a los tres puntos verticales y selecciona **"Instalar aplicación"** o **"Añadir a la pantalla de inicio"**.

## 🕹️ Cómo Jugar
- **Dibujar**: Toca y arrastra entre los puntos azules (anclajes) o nodos creados para poner un palo.
- **Borrar**: Usa la herramienta de la papelera para quitar palos si te equivocas.
- **Simular**: Pulsa el botón verde de "Simular" para soltar el peso.
- **Estrés**: Si los palos se ponen rojos, ¡están a punto de romperse!

## 🛠️ Tecnologías
- **Matter.js**: Motor de física 2D de alto rendimiento.
- **Vanilla JS/CSS**: Sin frameworks pesados para una carga instantánea.
- **PWA**: Soporte offline y acceso directo desde el escritorio.
